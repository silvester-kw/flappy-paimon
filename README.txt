**Cara Login ke Game (lebih lengkapnya ada di video demo)
1. Pengguna run file flappy_paimon.py
2. Pengguna mengisi id dan pass
3. Pengguna memilih save slot
4. Jika sudah register account, maka Anda bisa klik tombol "Start game", 
   jika belum register, Anda bisa menekan tombol "Register now" lalu tombol "Start game"
5. Untuk keluar game, tekan tombol esc pada keyboard

***Panduan di dalam Game (lebih lengkapnya ada di video demo)
1. Di main menu, pengguna dapat memilih 1 dari 3 fitur utama, yaitu Game, Gacha, dan Inventory
2. Klik tombol untuk memasuki laman
3. Jika memilih Game, maka Anda akan masuk ke mini game "Flappy Paimon". Mainkan game dengan menekan tombol space atau tombol mouse kiri untuk loncat. Di sini adalah tempat Anda menambahkan alat tukar (primogem)
4. Jika memilih Gacha, Anda dapat memilih untuk gacha 1x atau gacha 10x, banyak karakter/senjata yang didapat bergantung pada jumla wish
5. Jika memilih Inventory, Anda dapat melihat semua karakter/senjata yang sudah Anda dapatkan dari gacha beserta banyak dari karakter/senjata tersebut
6. Dari main menu anda dapat mereset akun dengan tombol "Reset" dan kembali ke laman Register/Login dengan tombol "Logout"
